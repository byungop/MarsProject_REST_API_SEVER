# MarsProject_REST_API_SEVER ( 아키텍쳐 및 소스로직 정리 중 )
- MARS 프로젝트는 저작권으로 인한 불편을 해소하기 위해 다양한 분야에 저작권 없는 이미지 데이터셋을 공급하려는 프로젝트 입니다.
- MARS 프로젝트 학습모델로 생성되는 데이터셋은 모두 저작권 없는 이미지이며, 학술 및 상업적 목적으로 자유롭게 사용할 수 있습니다.
